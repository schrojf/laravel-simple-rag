# Laravel Knowledge Provider

I am building **laravel-simple-rag** (Laravel Knowledge Provider) — a Laravel application that serves as an MCP server and a web-based knowledge manager where users can organize, manage (CRUD), and prepare their questions, snippets, context, documents, categories, tags, and other data to be provided to LLMs via MCP. LLMs can then write back their answers, scraped web data, and so on to the app via MCP.
